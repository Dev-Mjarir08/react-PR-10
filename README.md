# 📖 Overview

This project is a **React CRUD User Management Application** built to understand API integration and CRUD operations using React.

Users can create new records, fetch data from a REST API, display all users in a responsive table, and delete records instantly.

The backend is powered by **JSON Server**, while **Axios** handles API communication.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Dev-Mjarir08/PR-10.git
```

---

## Navigate

```bash
cd PR-10
```

---

## Install Packages

```bash
npm install
```

---

## Start JSON Server

```bash
npx json-server --watch db.json --port 3000
```

---

## Start React

```bash
npm run dev
```
